 ## COMO EJECUTARLO
1. Clonamos el repositorio
   
   	`git clone https://github.com/Geko222/Geko-Ig.git`

2. Nos colocamos en el repositorio
   
   `cd Geko-Ig`
   
3. Instalamos los requisitos
   
   	`python requisitos.py`
   
4. Iniciamos sesion
   
   	`python registro.py`
   
5. Ejecutamos la script y a darle
   
   	`python gekoig.py`
